# PDF Merger
A simple Python application that merges multiple PDF files into a single PDF document.

# Features
- Merge multiple PDF files
- Generate a single output PDF
- Easy-to-use script
- Fast processing

# Requirements
- Python 3.x
- PyPDF2 library

# Installation
Install the required library:
pip install PyPDF2

# How to Run
1. Place your PDF files in the project folder.
2. Update the file names inside:
pdf_files = [
    "file1.pdf",
    "file2.pdf"
]
3. Run:
pdf_merger.py
4. Output:
merged.pdf
